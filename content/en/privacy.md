## Cookies

From the [CNIL website](https://www.cnil.fr/fr/definition/cookie), a cookie is 

> A cookie is a small file stored by a server in the client (computer, telephone, etc.) of a user and associated with a web domain (i.e. in most cases with all the pages of the same website).  This file is automatically sent back during subsequent contacts with the same domain.

This website doesn't use any cookie.

## Collection of personal data

This website does not collect personnal data by any mean, other than the logs required by the French law as said in the following section.

This website may redirect you to tools that will collect personal data. Please refer to the Privacy Policy of the tools for more information.

## Logs

As a provider of an Internet Website, the law obliges that the logs of the web server are conserved for a duration of one year.

The logs are automatically generated by the web server and contain : 
* The source IP
* The date and time of request
* The HTTP request (method, requested resource and version of the protocol)
* The returned HTTP code of the request
* The size of the returned data
* The referer (which is a header of the request sent by the client)
* The User-Agent (which is also a header of the request). It allows to identify the browser used.

As described on the [apache2 website](https://httpd.apache.org/docs/2.4/fr/logs.html) the log line may look like : 
```
 127.0.0.1 - - [10/Oct/2000:13:55:36 -0700] "GET /apache_pb.gif HTTP/1.0" 200 2326 "http://www.example.com/start.html" "Mozilla/4.08 [en] (Win98; I ;Nav)"
```

(where the username as been replaced by - as there is no HTTP authentication on this website).

You can find more information on the [apache2 website](https://httpd.apache.org/docs/2.4/fr/logs.html) or by contacting privacy@nanoy.fr.
