In December 2022, I decided I wanted to give my co-workers an introduction tutorial to [Git](https://git-scm.com/). I prepared slides and some schemas (you can check the presentation [here](https://github.com/nanoy42/git-presentation)), but I felt it was missing something, some practical work. And then I had an idea: to create a Capture The Flag (CTF) challenge, where you have to use Git. The goal here will not be to gain some privileges on some machine but to find a string that has been deleted (but saved at some point in the version control), following some clues.
